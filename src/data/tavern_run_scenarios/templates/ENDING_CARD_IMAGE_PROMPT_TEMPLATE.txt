TAVERN RUN ENDING CARD IMAGE PROMPT TEMPLATE

How to use:
1) Generate all 5 family templates using the prompts below.
2) Export as PNG at 1200x630 (or 2400x1260 for 2x, then scale down).
3) Save into src/assets/tavern-run-cards/ as:
   - card-legendary.png
   - card-heroic.png
   - card-chaotic.png
   - card-cursed.png
   - card-pathetic.png
4) These are static background templates — the Canvas API stamps
   text (title, stats, player name, etc.) on top at runtime.

--------------------------------------------------
SHARED CONSTRAINTS (apply to ALL 5 prompts)
--------------------------------------------------
- Exactly 1200x630 pixels (16:9 landscape)
- No text, no logo, no watermark, no signature, no letters or numbers of any kind
- Include a semi-transparent aged parchment/scroll overlay band covering
  the center ~80% width and ~85% height of the image
- The parchment band should be yellowish-beige, ~70% opacity,
  with subtle paper grain texture and slightly rough/torn edges
- Leave the parchment band COMPLETELY EMPTY — no writing, no symbols,
  no icons, no decorations inside the parchment area
- The illustrated background should be visible around the edges
  and faintly through the parchment overlay
- The background scene should be darkened ~40% for text readability
- Each template has a unique decorative border AROUND the parchment band
  (not around the full image)

Negative guidance (append to all prompts):
blurry, low detail, overexposed, muddy colors, malformed anatomy,
extra limbs, any text, any letters, any numbers, any writing,
watermark, logo, signature, UI elements, frame outside the image
